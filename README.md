# MoMMI.

MoMMI is the bot that is used on the official [/vg/station13](http://ss13.moe/) and [Space Station 14](http://space-wizards.github.io/) Discord servers.

It runs on Python 3.6.


==what works==

wiggle
ss13 nudges
dm code compile
@mommi status
@mommi help
@mommi remind
wyci
@mommi magic8ball
@mommi pick(x,y)



==what doesnt work==
@mommi help github
